# Spring Cloud

实现微服务架构的框架

## 什么是微服务

将原来的单体应用拆分成微服务应用

一个工程拆分成多个工程，由多个工程各种完成不同的业务功能，共同构建一个大的系统架构

## 为什么要拆分

1、实现业务的解耦合

2、实现系统运行的分离

## 什么是 Spring Cloud？

Spring Cloud 是实现微服务架构的一种解决方案/框架，Spring 全家桶的一个分支，基于 Spring Boot 的

Spring -》Spring Boot -》Spring Cloud -》Spring Cloud Alibaba

## 注册中心

将服务拆分之后，把各个微服务分别注册到注册中心进行统一管理，方便接口调用。

注册中心就是用来管理各个微服务的。
